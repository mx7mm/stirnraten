## Why It Feels Good

- Clean, fullscreen-first experience
- Motion controls with per-round calibration
- Manual controls as reliable fallback
- Round setup before every game
- Big category library for instant replay value

## Gameplay

### Core Controls

- Tilt phone forward: correct
- Tilt phone backward: wrong/skip
- Button controls: `Richtig` and `Falsch / ueberspringen`
- Distinct feedback for correct vs wrong actions

### Round Setup

- Round length: `45`, `60`, `90` seconds
- Tilt sensitivity: `Niedrig`, `Mittel`, `Hoch`
- Pause and resume during active rounds

## Content

- 15 categories
- 622 terms total
- Popular themes like movies, stars, music, sports, gaming, Disney, and superheroes

## Tech

- Single-file app structure
- Plain HTML, CSS, and JavaScript
- No framework dependency
- Mobile-first layout and interactions

## Local Development

```bash
cd stirnraten
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Project Files

- `index.html` main app entry for GitHub Pages
